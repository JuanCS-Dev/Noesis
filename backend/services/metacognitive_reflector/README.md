# Metacognitive Reflector Service

Analyzes the system's own thought processes and performance to improve future decision-making.

## Quick Start

```bash
# Run service
python -m backend.services.metacognitive_reflector.main
```

## Architecture

Part of the Consciousness layer. Works with Maximus Core to provide self-reflection capabilities.

## API

- `POST /reflect`: Trigger a reflection cycle.

## Configuration

- `REFLECTION_INTERVAL`: Time between automatic reflection cycles.
